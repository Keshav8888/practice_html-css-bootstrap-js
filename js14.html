Statements in JavaScript ---------------------------------- Variables Data Types Operators Statements - Statements are used to control the execution flow. - JavaScript statements are classified into following types a) Selection Statements if, else, switch, case, default b) Iteration Statements for..in, for..of c) Looping Statements for, while, do while d) Jump Statements break, continue, return e) Exception Handling Statements try, catch, throw, finally Selection Statements: - They are used in decision making. - If condition is true the a set of statements to execute and another set of statements when condition is false. - Selection statement keywords if, else, switch, case, default 1. The "IF" selector a) Forward Jump b) Simple Decision c) Multi Level Decision d) Multiple Decision Forward Jump: It contains statements to execute only when condition is true.
